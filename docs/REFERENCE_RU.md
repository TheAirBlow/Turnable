# Справочник Turnable &nbsp;·&nbsp; [🇬🇧 EN](REFERENCE.md)
## Типы подключения
| Тип      | Описание                                                                                             |
|----------|------------------------------------------------------------------------------------------------------|
| `relay`  | Туннелирует трафик через TURN сервер платформы на шлюз сервера Turnable.                             |
| `direct` | Туннелирует трафик через TURN сервер платформы напрямую на конечный сервер. **⚠️ Не рекомендуется.** |
| `p2p`    | Скрывает трафик в поддельных трансляциях через SFU платформы. **⚠️ WIP**                             |


## Протоколы
| Протокол | Описание                                                                  |
|----------|---------------------------------------------------------------------------|
| `none`   | Без протокола вообще. **⚠️ Не рекомендуется и опасно использовать.**      |
| `dtls`   | Чистый DTLS. Простой, но легко обнаруживается. Только режим `relay`.      |
| `srtp`   | DTLS+SRTP. Имитирует настоящий медиатрафик. Принудительно в режиме `p2p`. |

## Транспорты
| Транспорт | Описание                                                                                                                      |
|-----------|-------------------------------------------------------------------------------------------------------------------------------|
| `none`    | Без транспортного протокола. Используй только для UDP маршрутов.                                                              |
| `kcp`     | [KCP](https://github.com/xtaci/kcp-go) - надежный и стабильный упорядоченный поток по UDP. Рекомендуется для TCP маршрутов.   |
| `sctp`    | [SCTP](https://en.wikipedia.org/wiki/Stream_Control_Transmission_Protocol) - поддерживается, но не идеален. Не рекомендуется. |

## Типы сокетов
| Сокет | Описание                                                                                                 |
|-------|----------------------------------------------------------------------------------------------------------|
| `tcp` | [TCP](https://en.wikipedia.org/wiki/Transmission_Control_Protocol) соединение (для VLESS, Trojan, и др.) |
| `udp` | [UDP](https://en.wikipedia.org/wiki/User_Datagram_Protocol) соединение (для WireGuard, Hysteria, и др.)  |

## Режимы шифрования
| Режим       | Описание                                              |
|-------------|-------------------------------------------------------|
| `handshake` | Шифрует только рукопожатие. Быстрее, меньше нагрузка. |
| `full`      | Шифрует весь трафик сквозным шифрованием.             |
