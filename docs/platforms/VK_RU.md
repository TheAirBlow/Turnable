# ВКонтакте &nbsp;·&nbsp; [🇬🇧 EN](VK.md)
Модуль платформы ВКонтакте позволяет получить токены аутентификации через систему видеозвонков VK и использовать их TURN и SFU инфраструктуру. Следуйте этому гайду чтобы безопасно всё настроить без риска блокировки вашего аккаунта.

## 1. Получите ID звонка
Чтобы минимизировать риск отслеживания запроса по вашему аккаунту VK, получите публичный ID звонка, [загуглив `"vk.com/call/join"`](https://www.google.com/search?q=%22vk.com%2Fcall%2Fjoin%22):

![Пример](https://files.catbox.moe/ic6sf9.png)

ID звонка - это строчка, которая идет после `https://vk.com/call/join/` в URL.

## 2. Решение капчи
Turnable пытается решать капчу автоматически, однако если автоматическое решение не работает, вам потребуется решить капчу вручную. Выполните следующие шаги:

### 2.1. Получите токены
Когда автоматическое решение не срабатывает, вы увидите в логах что-то на подобие этого:
```
2026-04-22 16:09:19.748 [INFO] manual captcha solve required userscript=http://localhost:1984/manual_captcha.user.js guide=http://localhost:1984/ url=https://vk.com/call/join/... timeout=10m0s
```

#### Шаг 1: Установите TamperMonkey
Установите расширение [TamperMonkey](https://addons.mozilla.org/ru/firefox/addon/tampermonkey/). Рекомендуется Firefox, так как он поддерживает расширения браузера даже на мобильных платформах, включая Android. Обязательно предоставьте TamperMonkey разрешение на работу в режиме инкогнито.

![Tampermonkey](https://files.catbox.moe/y5vlyg.png)

#### Шаг 2: Установите юзерскрипт
Скопируйте URL `userscript` из логов и откройте его в браузере. Вы должны увидеть диалоговое окно установки:

![Установка](https://files.catbox.moe/pq98f9.png)

Нажмите кнопку **Установить**, чтобы добавить юзерскрипт в ваш браузер.

### 2.2. Пройдите капчу
Перейдите по URL из логов и вручную пройдите капчу. Юзерскрипт:
- Автоматически нажмет кнопку **Присоединиться**
- Соберет все необходимые токены аутентификации
- Перенаправит вас с VK после завершения

Однако перед этим выполните следующее:
- **Используйте режим Инкогнито**, если вы уже авторизованы в VK. Это обязательно для предотвращения потенциальной блокировки.
- **Используйте режим Инкогнито на Android**, если у вас установлено приложение VK, чтобы предотвратить автоматическое перенаправление в приложение.
- **Включите вид для ПК** в вашем браузере, чтобы не быть перенаправленным на мобильную версию сайта.

## 3. Готово!
Токены аутентификации захвачены, и вы готовы использовать платформу VK с Turnable.
